## Quick Start: Compile PDFs

```powershell
# 1. Navigate to submission directory
cd "C:\Users\CCRN4\OneDrive\Desktop\TQS\TQS\HUBBLE_TENSION_NATURE_ASTRONOMY"

# 2. Generate figures (requires matplotlib, numpy)
python figures\make_all_figures.py

# 3. Copy/convert the MCMC corner figure
copy ..\fig48_mcmc_v9_corner.png figures\fig2_corner_mcmc.png

# 4. Compile main manuscript (2 passes for correct refs)
pdflatex -interaction=nonstopmode main_manuscript.tex
pdflatex -interaction=nonstopmode main_manuscript.tex

# 5. Compile supplementary information
pdflatex -interaction=nonstopmode supplementary_information.tex
pdflatex -interaction=nonstopmode supplementary_information.tex
```

**If `pdflatex` is not on PATH**, install MiKTeX or TeX Live:
- MiKTeX: https://miktex.org/download
- TeX Live: https://www.tug.org/texlive/
